# Security policy

## Supported version

Only the latest commit on `main` is supported. This is a static site with no first-party API, authentication, or stored customer data.

## Reporting a vulnerability

Do not open a public issue for a suspected vulnerability. Contact the repository owner through the [DevCalebR GitHub profile](https://github.com/DevCalebR) with the affected route, reproduction steps, and impact. Please allow reasonable time for review before disclosure.

Never include live credentials, personal data, or exploit payloads containing sensitive information in a report.
