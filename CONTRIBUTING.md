# Contributing

This repository is maintained as the RelayWorks company portfolio. Corrections, accessibility improvements, and reproducible bug reports are welcome.

1. Create a focused branch from `main`.
2. Install dependencies with `npm ci`.
3. Keep business claims supported by a linked project or product artifact.
4. Run `npm run validate`.
5. Open a pull request describing the user-visible change and validation result.

Do not commit credentials, analytics secrets, generated `out/` files, or invented client outcomes. Commercial service and brand changes should be discussed before implementation.
